- [OAuth and authentication](oauth-authentication.md) — Codex OAuth flow and API key authentication
- [Admin web UI](admin-web-ui.md) — built-in configuration dashboard
- [System tray](system-tray.md) — native Windows tray integration
- [Usage tracking](usage-tracking.md) — credit monitoring for OAuth accounts
